Introduction

Medical fraud causes significant financial losses each year. Detecting fraudulent activities effectively can save resources and improve healthcare services.

Objective

The project aims to develop robust machine learning models to identify and predict fraudulent activities in medical billing.

Data Overview

We used data from the Centers for Medicare and Medicaid Services, which included inpatient and outpatient details, provider data, and beneficiary details.

Methodology

Data Preparation

Merged various datasets to create a comprehensive dataset for analysis.

Data Preprocessing

Features were normalized and encoded to prepare for machine learning models.

Exploratory Data Analysis

Plot 1: Distribution of Claim Amounts

This plot shows the distribution of claim amounts, highlighting potential outliers and patterns of fraudulent claims.

Plot 2: Correlation Heatmap

The correlation heatmap helps identify relationships between different variables, which are crucial in predicting fraudulent activities.

We discovered key patterns and anomalies in the data that could indicate fraudulent activities.

Model Building

We implemented several models, including Logistic Regression, Decision Trees, and a Convolutional Neural Network.

Results

The Decision Tree model achieved the best performance with an accuracy of 82.92%.

Conclusion and Future Work

Future efforts will focus on integrating more data sources and improving model accuracy using advanced machine learning techniques.
